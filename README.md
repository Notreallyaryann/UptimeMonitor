# Uptime Monitor 🔍

A production-grade uptime monitoring service that checks if websites are up and sends email alerts when they go down. Built with Next.js, TypeScript, Prisma, and PostgreSQL.
## 📸 Architecture Overview

![Uptime Monitor Architecture](https://github.com/Notreallyaryann/UptimeMonitor/blob/main/Architecture.jpeg)

## ✨ Features

- ✅ **Monitor any website** - Add unlimited URLs to track
- ⏱️ **5-minute checks** - using cron-job.org
- 📧 **Email alerts** - Get notified immediately when a site goes down
- 🚦 **Smart alert throttling** - No spam, only notified on state changes
- ⚡ **Immediate verification** - New URLs are checked right after adding
- 📊 **Status dashboard** - See all your monitors at a glance
- 🔐 **User authentication** - Secure JWT-based login system
- 📈 **Check history** - View past uptime/downtime logs
